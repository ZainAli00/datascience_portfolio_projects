# 📊 Data Analyst Portfolio

A collection of 6 end-to-end data analysis projects demonstrating SQL, Python, Power BI/Tableau, and Excel skills across healthcare, finance, e-commerce, sports, property, and experimentation domains.

Built for the UK job market — all projects use UK-relevant datasets, metrics, and business contexts.

---

## Projects

### 1. 🏥 [NHS A&E Wait Times Analysis](./project-1-nhs-ae-analysis/)
Analysing 4 years of NHS England A&E performance data — seasonal patterns, regional disparities, and the 12-hour waits crisis.
- **Tools**: Python, SQL, Tableau
- **Skills**: Time-series analysis, data cleaning, public sector data

### 2. 💷 [UK Fintech Customer Churn Predictor](./project-2-fintech-churn/)
Predictive modelling to identify at-risk customers for a UK fintech, with risk segmentation and retention strategies.
- **Tools**: Python (Scikit-learn), SQL, Excel
- **Skills**: Machine learning, cohort analysis, business recommendations

### 3. 🛒 [E-commerce Sales & Marketing Dashboard](./project-3-ecommerce-dashboard/)
Multi-page dashboard design for a UK online retailer tracking revenue, conversion, and marketing channel ROI.
- **Tools**: Power BI, SQL, Python
- **Skills**: Data modelling, DAX, KPI dashboards, star schema

### 4. ⚽ [Premier League Performance Tracker](./project-4-premier-league-tracker/)
Deep-dive into the 2023/24 PL season with xG analysis, home/away splits, and referee impact stats.
- **Tools**: Python, SQL, Tableau
- **Skills**: Web scraping patterns, sports analytics, storytelling with data

### 5. 🏠 [UK Housing Market Explorer](./project-5-uk-housing-explorer/)
Exploring housing affordability across all UK regions by combining price data with income statistics.
- **Tools**: Python, SQL, Power BI
- **Skills**: Large dataset handling, geospatial analysis, joining public datasets

### 6. 🧪 [A/B Testing & Experimentation Report](./project-6-ab-testing-report/)
Full experiment lifecycle — power analysis, statistical testing, segment analysis, and a revenue-backed recommendation.
- **Tools**: Python (SciPy), SQL, Excel
- **Skills**: Hypothesis testing, confidence intervals, experiment design

## How to Use This Portfolio

Each project folder contains:
```
project-name/
├── README.md          ← Start here: overview, findings, screenshots
├── data/              ← Datasets (CSV) + SQLite databases
├── notebooks/         ← Python analysis scripts
├── sql/               ← Standalone SQL queries
└── visualisations/    ← Charts and dashboard screenshots
```

### Quick Start
```bash
git clone https://github.com/YOUR-USERNAME/data-analyst-portfolio.git
cd data-analyst-portfolio

# Install dependencies
pip install pandas numpy matplotlib seaborn scikit-learn scipy

# Run any project
cd project-1-nhs-ae-analysis
python notebooks/nhs_ae_analysis.py
```
